# University Food Map README

## 文件说明


当前目录下相关文件包括：

- `index.html`
- `设计文档.md`
- `README.md`

## 打开方式

直接双击 `index.html` 即可在浏览器中打开。

无需：

- 安装依赖
- 启动服务
- 使用 `package.json`

## 这一版的特点

相比原版，这一版重点优化了：

1. 输入框成为更明确的一级入口
2. 默认减少复杂度，高级筛选默认折叠
3. “AI” 文案更加克制，避免过度承诺
4. 保留地图和推荐逻辑，同时提升评审首屏理解效率

## 适合场景

- 产品题提交
- 评审快速预览
- 网址部署
- 与原版进行 A/B 展示对比

## 部署建议

因为它是纯静态页面，适合直接部署到：

- GitHub Pages
- Netlify
- Vercel

只要把 `index.html` 放在可访问目录中即可。
