# 使用说明

本项目为一个可直接本地打开的静态网页原型，适合产品题展示、上传或答辩演示。

## 文件说明

- `index.html`：网页主文件，已内置样式与交互逻辑
- `产品设计文档.md`：完整产品分析与功能设计文档
- `README.md`：当前说明文件

## 打开方式

直接双击 `index.html` 即可在浏览器中打开，无需安装依赖、无需启动服务。

## 原型特点

- 纯前端静态实现
- 不依赖外部地图或前端框架
- 包含“今天吃什么”推荐逻辑
- 包含高校联盟地图浏览
- 包含商户筛选、详情和学生榜单

## 适合的使用场景

- AI Coding 产品题提交
- 产品经理作品集展示
- 校园场景网页原型演示
